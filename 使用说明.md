# PDF数据提取使用说明

由于PDF格式复杂，自动提取可能不准确。请按以下步骤操作：

## 方法1：查看PDF格式（推荐第一步）

运行以下命令查看PDF的实际格式：

```bash
python3 view_pdf_format.py
```

这个脚本会显示PDF前5页的完整内容。**请将显示的内容发给我**，我会根据实际格式调整提取脚本。

## 方法2：手动提供数据

如果自动提取效果不理想，您可以：

1. **提供Excel文件**：将PDF内容整理成Excel，包含列：名称、性味、归经、功效、主治
2. **提供文本列表**：直接提供中药名称列表，我可以帮您查找或整理详细信息
3. **分段提供**：每次提供部分中药（如20-30个），我逐步帮您整理

## 方法3：参考标准数据库

我可以基于中药学教材的标准内容，为您创建一个包含常用中药的完整数据库（通常300-400个常用中药）。

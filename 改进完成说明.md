## 🎨 新的设计系统

### 颜色系统
```css
主色调: #4caf50 (温和绿色)
浅色: #81c784
深色: #388e3c
背景: #fafafa (浅灰)
卡片: #ffffff (白色)
```

### 间距系统
```css
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
```

### 圆角系统
```css
sm: 6px
md: 8px
lg: 12px
xl: 16px
full: 50%
```

### 阴影系统
```css
sm: 轻微阴影
md: 中等阴影
lg: 大阴影
hover: 悬停阴影（带绿色光晕）
```

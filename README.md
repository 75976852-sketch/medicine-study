# 🌿 中药背诵助手

一个帮助背诵中药的Web应用，类似于单词背诵软件，采用科学的学习和复习机制。

## 功能特点

### 📖 学习功能
- **每日设置背诵量**：可自定义每天学习的新中药数量（1-50个）
- **系统化学习**：按顺序学习中药，每次显示一个中药卡片
- **详细信息展示**：包含中药的别名、性味、归经、功效、主治、用法用量等信息
- **卡片翻转**：点击卡片可查看详细信息，再次点击翻回正面

### 🔄 复习功能
- **自动复习提醒**：第二天自动复习前一天学习的中药
- **智能过滤**：已掌握的中药不会被加入复习列表
- **复习反馈**：可以标记"已记住"或"未记住"，系统会根据反馈调整学习状态

### ✅ 掌握标记
- **已掌握标记**：学习或复习时可以标记中药为"已掌握"
- **自动排除**：已掌握的中药不会再出现在复习列表中
- **学习记录**：系统会记录每个中药的学习状态

### 📊 学习统计
- **实时统计**：显示今日新学数量、今日复习数量、已掌握数量
- **进度追踪**：显示总体学习进度（已学习/总数）
- **详细统计面板**：查看总体进度、已掌握数量、学习中数量、待复习数量

## 使用方法

### 基本操作

1. **设置每日目标**
   - 在"学习设置"面板中设置每天学习的新中药数量
   - 点击"保存设置"按钮保存

2. **开始学习新中药**
   - 点击"开始学习新中药"按钮
   - 点击卡片查看详细信息
   - 如果已经掌握，点击"已掌握"按钮；否则点击"下一个"继续

3. **开始复习**
   - 点击"开始复习"按钮（第二天会自动出现需要复习的内容）
   - 查看中药信息，评估自己的记忆情况
   - 点击"已记住"或"未记住"进行反馈
   - 点击"下一个"继续复习

4. **查看统计**
   - 点击"学习统计"查看详细的学习数据

### 数据存储

所有数据存储在浏览器的 localStorage 中，包括：
- 每日学习目标
- 已学习的中药ID列表
- 已掌握的中药ID列表
- 学习历史记录（按日期）
- 复习历史记录（按日期）

### 重置数据

如果需要重新开始学习，可以点击"重置数据"按钮，会清除所有学习记录。

## 技术实现

- **纯前端应用**：使用HTML、CSS、JavaScript实现
- **本地存储**：使用 localStorage 保存学习数据
- **响应式设计**：适配不同屏幕尺寸
- **现代化UI**：采用渐变色彩和流畅动画

## 中药数据库

应用内置30个常用中药的详细信息，包括：
- 人参、黄芪、当归、白术、甘草
- 枸杞子、熟地黄、白芍、川芎、茯苓
- 陈皮、桂枝、柴胡、半夏、生姜
- 大枣、山药、麦冬、五味子、党参
- 丹参、黄连、黄芩、金银花、连翘
- 板蓝根、板栗、薄荷、牛蒡子、菊花

每个中药包含：名称、别名、性味、归经、功效、主治、用法用量等信息。

## 使用建议

1. **每日坚持**：设置合理的每日学习量，坚持每天学习
2. **及时复习**：第二天及时复习前一天的内容，加深记忆
3. **诚实标记**：客观评估自己的掌握情况，已掌握的中药要标记清楚
4. **循序渐进**：不要设置过大的每日目标，循序渐进更有利于长期坚持

## 注意事项

- 数据存储在浏览器本地，清除浏览器数据会丢失学习记录
- 建议在同一台设备上使用，确保学习记录的连续性
- 如果需要在不同设备间同步，可以考虑导出/导入数据功能（后续可添加）

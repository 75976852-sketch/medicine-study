# 🚀 5分钟快速部署 - Vercel 方案（最简单）

## 📌 步骤总结（5步搞定）

### ✅ 第1步：准备代码
确保你的文件夹有这些文件：
- index.html ✅
- style.css ✅  
- script.js ✅

### ✅ 第2步：注册账号
1. 打开 https://vercel.com
2. 点击右上角 "Sign Up"
3. 选择 "Continue with GitHub"（推荐）

### ✅ 第3步：上传到 GitHub
1. 打开 https://github.com，登录
2. 点击右上角 "+" → "New repository"
3. 仓库名：`medicine-study`（随便起）
4. 选择 Public
5. 点击 "Create repository"
6. 点击 "uploading an existing file"
7. **把项目文件夹里的所有文件拖进去**
8. 点击 "Commit changes"

### ✅ 第4步：部署到 Vercel
1. 在 Vercel 点击 "Add New..." → "Project"
2. 找到你的 GitHub 仓库（medicine-study）
3. 点击 "Import"
4. 直接点击 "Deploy"（不用改设置）
5. 等待1-2分钟

### ✅ 第5步：完成！
部署成功后会显示地址，比如：
```
https://medicine-study-xxxxx.vercel.app
```

**点击这个地址就能访问你的网站了！** 🎉

---

## 🎁 额外福利：绑定自己的域名

如果想让别人用 `medicine.yourname.com` 访问：

1. 在 Vercel 项目页面 → "Settings" → "Domains"
2. 输入你的域名
3. 按照提示配置 DNS（添加 CNAME 记录）
4. 等待5-30分钟生效

---

## 💡 更新网站

以后想更新网站：
1. 在 GitHub 更新代码
2. Vercel 会自动重新部署
3. 等1-2分钟，网站就更新了！

## 🚀 最快部署方法（推荐新手）

### 使用 Vercel（5分钟搞定）

1. **注册 Vercel**：https://vercel.com（用 GitHub 账号）
2. **上传代码到 GitHub**：创建仓库，上传文件
3. **在 Vercel 导入项目**：选择 GitHub 仓库，点击部署
4. **完成！** 获得网站地址

详细步骤请查看 [`一键部署-Vercel.md`](./一键部署-Vercel.md)

## 📦 项目文件说明

部署时需要的文件：
- ✅ `index.html` - 主页面
- ✅ `style.css` - 样式文件
- ✅ `script.js` - 功能代码
- ✅ `package.json` - Node.js 配置（可选）
- ✅ `server.js` - Node.js 服务器（可选）

其他文件可以不上传：
- 📝 `.md` 文档文件
- 🐍 `.py` Python 脚本
- 📄 `.docx`、`.pdf` 等资源文件

## ❓ 常见问题

### Q: 我是完全的小白，应该选哪个？
**A:** 强烈推荐 **Vercel**！最简单，5分钟搞定，完全免费。

### Q: 没有域名可以部署吗？
**A:** 可以！Vercel 会给你一个免费地址，比如 `your-app.vercel.app`

### Q: 部署后数据会丢失吗？
**A:** 不会。数据存在用户浏览器里（localStorage），不影响服务器。

### Q: 可以多人使用吗？
**A:** 可以！支持多用户，每个用户数据独立存储。
